# 地点关键词输入提示

## 接口信息

- **接口地址**: `/proxy/suggestion`
- **请求方法**: `GET`
- **认证方式**: JWT Token（在 Header 中携带）
- **Content-Type**: `application/json`

## 功能说明

用于获取输入关键字的补完与提示，帮助用户快速输入。本接口通过调用腾讯地图地点输入提示服务实现，适用于实现 Autocomplete（自动完成）的效果。

## 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| keyword | string | 是 | 搜索关键词，最多支持96个字符（每个英文字符占1个，中文占3个） |
| region | string | 否 | 限制城市范围，根据城市名称限制地域范围，如"广州"；支持设置行政区划编码 |

### 请求示例

```
GET /proxy/suggestion?keyword=美食&region=北京
Authorization: Bearer <your_jwt_token>
```

## 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | number | HTTP状态码，200表示成功，其他为腾讯地图返回的状态码 |
| status | number | 腾讯地图API状态码，0为成功，其他为异常 |
| message | string | 状态说明信息 |
| request_id | string | 本次请求的唯一标识 |
| count | number | 结果总数（本服务一个查询条件最多返回100条结果） |
| data | array | 提示词数组，每项为一个POI对象 |

### data 数组中的 POI 对象字段

| 参数名 | 类型 | 说明 |
|--------|------|------|
| id | string | POI唯一标识 |
| title | string | 提示文字（地点名称） |
| address | string | 地址 |
| category | string | POI分类 |
| location | object | 提示所述位置坐标 |
| location.lat | number | 纬度 |
| location.lng | number | 经度 |
| adcode | number | 行政区划代码 |
| province | string | 省 |
| city | string | 市 |
| district | string | 区/县 |
| type | number | POI类型：0:普通POI / 1:公交车站 / 2:地铁站 / 3:公交线路 / 4:行政区划 |

### 错误响应示例

**400 错误 - 请求参数错误或腾讯地图API错误**

```json
{
  "code": 400,
  "message": "腾讯地图API错误: 300 - Invalid key"
}
```

**503 错误 - 腾讯地图服务不可用**

```json
{
  "code": 503,
  "message": "无法连接到腾讯地图服务，请稍后重试"
}
```

## 错误码说明

| status | 说明 |
|--------|------|
| 0 | 成功 |
| 300 | 签名校验失败 |
| 310 | 权限校验失败 |
| 320 | 服务未开通 |
| 330 | 服务调用超限 |
